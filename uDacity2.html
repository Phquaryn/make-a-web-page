The World Wide Web is made up of documents, mostly html, linked together

The first major piece of the web is actually three pieces: you, your computer and the browser. Simply, you go to a page with your computer using a browser. The Internet is made up of different types of documents, but the main type is HTML, or HyperText Markup Language documents. The links between the 30 Billion html and other documents are what make the web "web-like". Other major pieces of the web are http protocol and the servers.


Use a consistantly styled HTML document coupled with CSS to create a well structured page

HyperText Markup Language is the standard markup language used to create and structure web pages. HTML controls the tree-like structure of a page, or "DOM" (Document Object Model). Per Wikipedia, "A tree structure... is a way of representing the hierarchical nature of a structure in graphical form" and is "generally upside down compared to an actual tree".
Like HTML, CSS is a language. "Cascading Style Sheets - style sheet language used for describing the look and formatting of a document written in a markup language." CSS controls the style of a page. The tutorial touches on the Chrome Dev Tools to check out/play with CSS elements and the office hours q&a goes into using it in more depth. avoiding repitition with CSS


The Internet is made up of boxes - remember this when creating html page layouts

Elements on a web page are all boxes. Boxes are easy to relocate. Name your tags something descriptive too help keep your boxes organized, like class ="screenshot" or div class = "title" (or delicious ice cream flavors, to help you choose and organize colors for a random color scheme.).
